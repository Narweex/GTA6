# Please adhere to the following directory structure 

```
res://
├── .godot/                  # Auto-generated by Godot (Do not touch / ignore in Git)
├── common/                  # Shared resources used globally across the game
│   ├── audio/               # Background music, global UI sound effects
│   ├── ui/                  # Global UI themes, fonts, generic buttons
│   └── shaders/             # Global shader materials
├── components/              # Game objects and logic (Organized by feature)
│   ├── player/
│   │   ├── player.tscn
│   │   ├── player.gd
│   │   └── player_spritesheet.png
│   ├── enemies/
│   │   ├── goblin/
│   │   │   ├── goblin.tscn
│   │   │   └── goblin.gd
│   └── main_menu/
│       ├── main_menu.tscn
│       ├── main_menu.gd
│       └── title_logo.png
├── levels/                  # Your game worlds / scenes
│   ├── level_base.gd        # Base script that all levels inherit from
│   ├── level_01/
│   │   ├── level_01.tscn
│   │   └── level_01.gd
│   └── level_02/
└── autoload/                # Global scripts (Singletons) for state/data
	├── game_manager.gd
	└── save_system.gd
```

#Conventions
- Please use snake style coding style
